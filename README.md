Compile the program by typing `make` into the terminal.

The program can be ran with ./main command, but a basic script handling input arguments is provided at `handling.sh`.
To use it, make sure it has execution permission `chmod +x handling.sh`.
Please modify the input arguments. By default, the input files are stored in the `inputs/` directory, while the outputs are saved in `outputs/`.

Some basic gnuplot scripts are also provided for plotting in `plotting/`

For more info, please check out the `ProjectReport.pdf`

Have fun!
